# TermuxForge AI Promotional Web Page

## Overview
This directory contains promotional web pages for TermuxForge AI, emphasizing its public service mission of democratizing visual AI education.

## Files Created

1. **`public_service_promo.html`** - Comprehensive promotional page
   - Full-featured website with animations
   - Detailed feature explanations
   - Public service mission focus
   - Responsive design
   - Interactive elements

2. **`index.html`** - Simplified landing page
   - Clean, mobile-friendly design
   - Quick overview of key features
   - Direct download links
   - Minimalist approach

## How to Use

### Local Testing
```bash
# Using Python's HTTP server
python3 -m http.server 8000
# Then visit http://localhost:8000/public_service_promo.html

# Or using PHP
php -S localhost:8000
```

### Deployment Options
1. **GitHub Pages** - Free hosting for the project
2. **Netlify/Vercel** - Easy deployment with CI/CD
3. **Static hosting** - Any web server that serves HTML
